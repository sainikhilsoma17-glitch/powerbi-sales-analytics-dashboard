# powerbi-sales-analytics-dashboard
Interactive Power BI dashboard analyzing sales performance using Power Query, DAX, and Star Schema modeling.
Power BI Sales Dashboard Project
Project Overview
This project demonstrates the creation of an interactive Power BI dashboard used to analyze sales performance, shipment distribution, and product profitability.
The dashboard helps stakeholders monitor key metrics such as revenue, profit percentage, shipment count, and product performance across different regions.

Dataset
The dataset used in this project contains:
Sales transactions
Product information
Shipment details
Geographic sales data
Files included in this repository:
dataset.xlsx
dataset.xml

Data Preparation
Data preparation steps performed in Power BI:
Imported dataset into Power BI
Cleaned data using Power Query
Removed unnecessary columns
Created calculated columns
Handled missing values
Formatted data types

Data Modeling
Implemented Star Schema data model.
Fact Table:
Sales / Shipments
Dimension Tables:
Products
Locations
Calendar
Relationships were created between tables to enable efficient analysis.

DAX Calculations
Several DAX measures were created including:
Total Sales
Total Profit
Shipment Count
Profit Percentage
Year-over-Year (YoY) Sales
Top Products Analysis
DAX concepts used:
Evaluation Context
VAR variables
Time Intelligence
